PalCalc breeding solver
-----------------------
- tylercamp/palcalc v1.17.6, pinned at commit
  8b7e2f779e47fddae16ddcb973e828ba20c02b80: MIT.
- PalPanel references only PalCalc.Model and PalCalc.Solver from the
  cross-platform `palcalc-bridge`; the WPF UI, GraphSharp, and Windows-only
  save reader are not linked. The complete license is included as
  licenses/PalCalc-MIT.txt.
- The 115-entry standard passive-skill Chinese catalog is reproducibly
  generated from PalCalc.Model/db.json v26 by
  scripts/generate-palcalc-passives.go under the same MIT license.

Runtime-downloaded PalDefender
------------------------------
- PalDefender, copyright Ultimeit: MIT. The complete license is included as
  licenses/PalDefender-MIT.txt.
- PalPanel does not bundle PalDefender binaries. During an explicitly requested
  install or update, it downloads d3d9.dll and PalDefender.dll from the latest
  stable official GitHub Release and verifies the published SHA-256 digests.
  A digest-verified PalDefender.zip is used only when the direct DLL assets are
  not present in that Release.

Runtime community icon references
---------------------------------
- The player center can request Pal icons from the public GitHub Raw paths in
  bingyouxue/palworld-server-tool-gm. These images are not copied into or
  redistributed with PalPanel packages; failed requests fall back to Lucide.
- Palworld names, characters, and artwork remain property of Pocketpair, Inc.
  Referencing a remote image does not relicense that artwork under PalPanel's
  GPL.
