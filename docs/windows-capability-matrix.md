## Remaining Live Validation

The retained evidence above proves the native SteamCMD/game install, Shipping-Cmd lifecycle,
authenticated SteamCMD Workshop download, package smoke, and native UE4SS/PalDefender load
logs. Before declaring the Windows release complete, follow `docs/windows-validation.md` to
repeat those checks against the final candidate, complete the packaged UI Workshop flow,
scan one real manually installed Mod, and perform GM actions only with an explicitly
disposable test player. If an external platform prevents a live check, retain a redacted
reproducible error and command instead of claiming success.
